# Medicine Finder

Невеликий Python-додаток для пошуку препаратів серед аптек.

---

## Можливості

* Знаходження аптек що мають певний препарат

  * **unit tests** (через unittest)

---

## Завантаження

1. Схиляти репозиторій:

``` bash
git clone https://github.com/artemijMoskowsky/imp_l6
```

2. Перейти до папки проекту:

``` bash
cd imp_l6
```

---

## Запуск

Запуск основної програми:

``` bash
python main.py
```

Приклад висновку:

```
Аптеки що мають препарат: Аптека 2, Аптека 4
```

---

## Запуск тестів

Усі тести запускаються автоматично разом із програмою:

``` bash
python main.py
```

## Приклади використання

```python
pharmacies = [
  {
      "name": "Аптека 1",
      "medicines": ["Аспірин", "Парацетамол"]
  },
  {
      "name": "Аптека 2",
      "medicines": ["Ібупрофен"]
  },
  {
      "name": "Аптека 3",
      "medicines": ["Аспірин", "Нурофен"]
  },
  {
      "name": "Аптека 4",
      "medicines": ["Парацетамол", "Ібупрофен"]
  }
]
print("Аптеки що мають препарат:", ", ".join(find_available_medicines(pharmacies, "Ібупрофен")))
```

## Вимоги

* Python 3.8+

## Нотатки

* `find_available_medicines` - Функція для визначеня у яких аптеках є препарат. Повертає список назв аптек.
* Проект зроблено як навчальний приклад (з тестами)
